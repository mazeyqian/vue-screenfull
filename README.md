A Vue3 component for Fullscreen.

## Install

You can get vue-screenfull via [npm](https://www.npmjs.com/package/vue-screenfull).

```
npm install vue-screenfull --save
```

## Usage

Foo description.

```
import { foo } from 'vue-screenfull';

foo();
```

Bar description.

```
import { bar } from 'vue-screenfull';

bar();
```

## Develop

```
# dev
npm run dev

# build
npm run build

# documentation
npm run docs
```
